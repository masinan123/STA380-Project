This vignette serves as an example of how to use the implemented functions to perform a two-group permutation test for comparing demographic indicators across development groups using country-level data.

Specifically, the following examples demonstrate:

  • two-group permutation test based on the difference in means perm_test_two_group(),

  • computation of permutation-based p-values perm_p_value(), and

  • visualization of permutation distributions with the observed statistic plot_perm_dist().

In particular, the examples focus on country-level estimates from the World Population Prospects 2024 dataset and illustrate how permutation-based inference can be used to assess whether observed differences between development groups are statistically significant.

Running the Permutation Test

We run a two-sided permutation test using the difference in means (More developed - Less developed) as the test statistic.

Hypotheses

Let \(\mu_M\) be the mean outcome for More developed and \(\mu_L\) for Less developed.

\[ H_0: \mu_M = \mu_L \]

\[ H_A: \mu_M \ne \mu_L \]

We use the observed test statistic:

\[ T_{\text{obs}} = \bar{X}_M - \bar{X}_L \]

and generate a permutation distribution by shuffling the group labels \(B\) times.

res <- perm_test_two_group_mean(
  df = test_df,
  B = 2000,
  seed = 1,
  alternative = "two.sided"
)

res$observed
#> [1] 10.87906
res$p_value
#> [1] 0

Visualizing the Permutation Distribution

We plot the permutation distribution and highlight the observed test statistic with a vertical line.

plot_perm_dist(res, bins = 40)

Conclusion (reject vs. fail to reject)

We conclude using a significance level α=0.05.

alpha <- 0.05

if (res$p_value < alpha) {
  cat("Since p <", alpha, ", we reject $H_0$.\n\n")
  cat("The observed difference in means is unlikely under random label assignment, ",
      "providing statistical evidence that the indicator differs between development groups in 2023.\n")
} else {
  cat("Since p \\ge", alpha, ", we fail to reject $H_0$.\n\n")
  cat("The observed difference in means is consistent with the permutation distribution ",
      "under exchangeability; there is insufficient statistical evidence of a difference between development groups.\n")
}

Since p < 0.05 , we reject \(H_0\).

The observed difference in means is unlikely under random label assignment, providing statistical evidence that the indicator differs between development groups in 2023.
