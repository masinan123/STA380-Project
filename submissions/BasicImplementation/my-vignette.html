Permutation test vignette: WPP 2024 development groups

This vignette demonstrates how to use the package functions to perform a two-group Monte Carlo permutation test comparing demographic indicators across development groups using country-level data.

The workflow shown includes:

  • preparing a clean two-column dataset with perm_data_extract(),

  • computing a two-sample Kolmogorov–Smirnov (KS) statistic with perm_stat_ks(),

  • running a permutation test with perm_test_two_group_ks(), and

  • visualizing the permutation distribution with plot_perm_dist().

Running the Permutation Test (KS statistic)

We run a two-sided permutation test using the KS statistic, which measures the maximum difference between the empirical CDFs of the two groups.

Hypotheses

Let \(F_M(x)\) be the distribution of the More developed group and \(F_L(x)\) be the distribution of the Less developed group.

\[H_0: F_M(x) = F_L(x)\] \[H_A: F_M(x) \neq F_L(x)\]

The observed test statistic is the Kolmogorov-Smirnov distance, defined as:

\[D_{obs} = \sup_x |F_M(x) - F_L(x)|\] We generate a permutation distribution by shuffling the group labels 𝐵 times.

res <- perm_test_two_group_ks(
  df = test_df,
  B = 2000,
  seed = 1,
  alternative = "two.sided"
)

res$observed
#> [1] 0.738161
res$p_value
#> [1] 0.0004997501

Notes on the p-value

The function computes a Monte Carlo permutation p-value using the standard adjustment:

\(p = \frac{\#\{\text{extreme permutations}\} + 1}{B + 1}\)

This prevents reporting a p-value of exactly 0 when 𝐵is finite.

Visualizing the Permutation Distribution

We plot the permutation distribution and highlight the observed KS statistic with a vertical line.

plot_perm_dist(res, bins = 40)

Interpretation: The observed KS statistic lies far in the right tail of the permutation distribution, indicating that the observed difference between the two groups is much larger than would typically arise under random label assignment.

Inspecting Outcome Distributions by Group


outcome <- test_df$outcome
group <- test_df$group

groups <- as.character(group)
groups <- unique(groups)
  
if (length(groups) != 2) {
    stop("group must have exactly two levels. ")
}
xA <- outcome[group == groups[1]]
xB <- outcome[group == groups[2]]
  
ecdf_A <- stats::ecdf(xA)
ecdf_B <- stats::ecdf(xB)
  
plot(ecdf_A, col="blue", lwd=1,
       main="ECDF Comparison between two groups",
       xlab="Outcome",
       ylab="Cumulative Probability")
  
lines(ecdf_B, col="yellow", lwd=1)
  
legend("bottomright",
         legend=c("More developed","Less developed"),
         col=c("blue","yellow"),
         lwd=2)

Conclusion (reject vs. fail to reject)

We conclude using a significance level α=0.05.

alpha <- 0.05
p_value <- round(res$p_value, 4)

perm_test_conclusion(p_value, alpha)

Since 5e-04 (p-value) < 0.05 (alpha), we reject H0. The observed KS statistic is unlikely under random label assignment, providing statistical evidence that the indicator distribution differs between development groups in 2023.
